# IScream
[![All Contributors](https://img.shields.io/badge/all_contributors-5-orange.svg?style=flat-square)](#contributors)

*Iscream是一个有声读物软件，基础调用了讯飞的中文语音接口，经过Tensorflow分析段落语速语义，生成接近人声情感语音*

### 即将上线

- 关键词搜索
- 自定义上传文本
